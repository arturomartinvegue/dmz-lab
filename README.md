# Proyecto - Configuración de una red DMZ.
El objetivo del proyecto ha sido crear una red DMZ, una NAT y aplicar reglas ACL en un router Cisco.

Los contenidos del repositorio son:
- Archivo .pka con el proyecto finalizado con el programa Cisco Packet Tracer.
- Archivo README.md con explicación breve del objetivo del proyecto y el contenido del repositorio.
- Directorio informe con los resultados del laboratorio y las pruebas realizadas junto con las evidencias gráficas.
